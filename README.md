# 🎵 Musicify

A modern music streaming web application built with **PHP, HTML, CSS, and JavaScript**, powered by a Music API. Musicify allows users to discover trending songs, explore artists and albums, and play music through an intuitive and responsive interface.

---

## ✨ Features

- 🎧 Browse Trending Songs
- 🔍 Search Songs, Artists, and Albums
- 💿 Explore Popular Albums
- 🎤 View Top Artists
- ▶️ Built-in Music Player
- ❤️ Favorite Songs
- 🕒 Recently Played
- 👤 User Authentication
- 📱 Responsive Design
- 🌙 Modern Dark Theme

---

## 🚀 Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/musicify.git
```

### 2. Open the project

Move the project to your web server.

**XAMPP**

```text
C:\xampp\htdocs\musicify
```

**Laragon**

```text
C:\laragon\www\musicify
```

---

### 3. Start Apache

Open XAMPP or Laragon and start:

- Apache
- MySQL (optional)

---

### 4. Open the application

```
http://localhost/musicify
```
